# Happy/Sad Classifier

A simple **image classification web app** built using **Keras** and **Streamlit** that can classify images into **Happy** or **Sad** categories.  

---

## Features

- Upload an image and get real-time prediction.
- Preprocessing ensures any uploaded image is resized and converted to RGB.
- Streamlit frontend for easy interaction.
- Binary classification: Happy / Sad.
- Shows raw prediction score along with class label.
